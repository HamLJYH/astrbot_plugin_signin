# AstrBot 签到插件 

一个功能完善的 AstrBot 签到插件，支持每日签到、连续签到、积分排行、积分商店、道具系统、积分转账等功能。

## 功能特性

- ✅ **每日签到** - 每日签到获取基础积分
- 🔥 **连续签到** - 连续签到获得额外积分加成
- 🎲 **幸运抽奖** - 签到时有概率触发随机额外积分奖励
- 🏆 **积分排行** - 查看全服积分排行榜
- 📊 **签到统计** - 查看个人签到详情、连续天数、累计签到等
- 🛒 **积分商店** - 用积分购买各种道具
- 🎮 **道具系统** - 占卜卡、改名卡、补签卡、抽奖券、幸运符等
- 💸 **积分转账** - 用户之间可以转账积分（通过QQ号）
- 💾 **数据持久化** - 数据自动保存到本地，重启不丢失
- ⚙️ **灵活配置** - 通过 WebUI 配置各项参数
- 🛡️ **错误处理** - 统一的错误处理机制，友好提示

## 安装方法

1. 将本插件文件夹放入 `AstrBot/data/plugins/` 目录下
2. 确保目录名为 `astrbot_plugin_signin`
3. 在 AstrBot WebUI 中重载插件
4. （可选）在插件配置页面调整参数

## 目录结构

```
astrbot_plugin_signin/
├── main.py              # 插件主文件
├── metadata.yaml        # 插件元数据
├── _conf_schema.json    # 配置项定义
├── requirements.txt     # 依赖声明（本插件无额外依赖）

```

## 使用指令

### 签到相关
| 指令 | 说明 |
|------|------|
| `/签到` | 每日签到，获取积分 |
| `/签到信息` | 查看个人签到详情 |
| `/签到排行` | 查看积分排行榜 |

### 商店与道具
| 指令 | 说明 |
|------|------|
| `/商店` | 查看积分商店商品 |
| `/购买 <编号>` | 购买商品（如 `/购买 1`） |
| `/占卜` | 使用占卜卡查看今日运势 |
| `/改名 <名称>` | 使用改名卡修改显示名 |
| `/补签` | 使用补签卡补签昨天 |
| `/抽奖` | 使用抽奖券参与积分抽奖 |

### 其他
| 指令 | 说明 |
|------|------|
| `/转账 QQ号 金额` | 转账积分给其他用户（通过QQ号） |
| `/签到帮助` | 显示帮助信息 |

## 配置项

在 AstrBot WebUI → 插件 → 签到系统 → 配置 中可调整：

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 基础签到积分 | 10 | 每日签到基础积分 |
| 连续签到加成 | true | 是否启用连续签到奖励 |
| 连续签到加成比例 | 0.1 | 每天额外增加 10% |
| 最大连续签到加成倍数 | 2.0 | 加成上限（最多 2 倍） |
| 排行榜显示数量 | 10 | 积分排行显示前几名 |
| 签到重置时间 | 5 | 每日凌晨 5 点重置 |
| 启用排行榜 | true | 是否显示排行榜 |
| 启用幸运抽奖 | true | 是否触发随机奖励 |
| 幸运抽奖最大额外积分 | 50 | 随机奖励上限 |
| 启用积分商店 | true | 是否启用商店功能 |
| 启用积分转账 | true | 是否允许转账 |

## 数据存储

签到数据存储在 `AstrBot/data/plugin_data/astrbot_plugin_signin/signin_data.json` 中。

## 开发规范

本插件遵循 [AstrBot 插件开发规范](https://docs.astrbot.app/dev/star/plugin-new.html)：
- 使用 `dataclass` 管理配置
- 统一的错误处理装饰器
- 类型注解
- 数据持久化到 `data` 目录
- 使用 `yield` 返回消息
